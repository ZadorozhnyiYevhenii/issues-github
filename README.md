## Installation and Usage `React Vite`

1. Install dependencies:

   - `npm install` for client and server parts

2. Start the application:
   - `npm run dev` for running the app

3. To build project:
   - `npm run build`

4. To start the production build:
   - `npm run start`

## Technologies

- React.js (14.1.4)
- TypeScript
- Ant design

## API
